SQL(Structure query language):
it is used to communicate with the databases

mainly it used to manage the data whether you want to create,read,update and even delete

sql provides different commands to perform the operations

DDL(data definition language)
DML(data manipulation language)
TCL(transaction control language)
DCL(Data control language)
DQL(Data query language)

what the purpose of learning sql:

data science & analytics:

it helps to query huge amount of the data and help to generate reports and insights 

website development:

it is very helpful to manage the data(website,transactions etc)

SQL database:

it is nothing but the process where we can create and manage the database

create command to create a database

CREATE database database_name

if you want to see what kind of databases available

show databases

if you want to use any one specific database

use database_name

if you don't want to use that database:

drop database database_name


1.DDL(data definition language):

it is used to define the structure of the tables

CREATE--->used to create table/database
ALTER---->used to modify the table
DROP---->used to delete the table/database
TRUNCATE--->used to remove the records but the structure remains

create database database_name

create table table_name(col1 dt,col2 dt----)

2.DML(data manipulation language):

it is used to manipulate the data

INSERT----->it is used to add the data
UPDATE----->it is used to edit the data
DELETE------>it is used to remove the data

3.DCL(data control language):

it is mainly used to control the permissions of the database

GRANT----->which is used to give the permissions
REVOKE----->it is used to remove the permissions

4.TCL(transaction control language):

it is used to manage transactions in the database

COMMIT----->it is used to save the changes permanently
ROLLBACK---->it is used to undo the changes
SAVEPOINT---->which is used to create the rollback point

5.DQL(data query language):

SELECT---->it is used to fetch the data
HAVING clause
GROUPBY clause
ORDERBY clause

How sql works:

1.input---->user sends the query what to select and it will parse the data

2.optimization---->used to run the query faster

3.execution--->where the database will run the query

4.output---->where the result send to the user

database: A structured collection of data stored in tables in the

Joins: which is used to combine the different tables where we have

SQL operators:

where it is used to perform operations on the sql data

1.Arithmetic operators:

it is used to perform mathematical operations

2.Comparision operators:

it is used to compare two expressions

3.Logical operators:

it is used to combine and manipulate the conditions

4.bitwise operators:

it is used to perform on binary values

5.special operators:

it is used to perform on specific condition and for filtering the data

like:between/and,in,exits,not exits,like etc
